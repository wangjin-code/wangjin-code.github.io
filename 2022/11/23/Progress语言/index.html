常用语句

​ 定义变量：Define variable var_name [As Data_Type format format].

​ init value Label “ABC”.

​ Like Field.

语法说明：

1、var1,var2,var3,var4,var5都是变量名，变量名不要与系统关键字重复﹐字符或者下划线开头﹐比如strModel，_Model 。
as后面接变量类型﹐比如integer ， date , logical ,character等。
label就是后续程序中对变量的描述﹐相当于一个标签；initial指定变量的初始值
2、PROGRESS支持语法简写·def和var都是简写﹐def是define的简写﹐var是variable的简写.。
3、like和as的不同点是︰as后面直接说明变量类型﹐而like后面跟另外一个变量或者数据字段。
4、extent ,表示该变量是数组变量。Extent 3表示该变量为一个三维数组。
5、format “x(5)”变量格式。
指定变量的格式一个最大的好必就是预留宽度﹐这个对变量的输人或者报表的输出很有用·比如年份的宽度一定是4位的﹐那么你就可以指定格式format “9999”。
6、PROGRESS程序每行结束必须有个“.”号。

​ 赋值变量：Assign var_name = value.

​ 显示：Display var_name.

​ 使用流程: Define variable –> Assign variable –>Display variable

​ 先定义类型、长度、格式，再进行声明赋值，再使用

​ 用户输入: Set var_name.
